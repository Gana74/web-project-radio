Landing page radio

## Описание

Сайт, включающий адаптивный дизайн, мультимедийный контент и интерактивные элементы. Проект использует Swiper.js для слайдеров и другие современные библиотеки.

## Используемые технологии

- HTML5
- CSS3
- JavaScript
- Swiper.js
- Normalize.css
